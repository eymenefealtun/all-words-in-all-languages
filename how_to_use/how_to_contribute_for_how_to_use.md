# How to contribute?

1. [Fork](https://github.com/eymenefealtun/all-words-in-all-languages/fork) the repository.

2. Create a folder for your language [here](https://github.com/eymenefealtun/all-words-in-all-languages/tree/main/) with the name of your language 
				 
3. Create a file for your language inside of the folder you created with the name of `YourLanguage.txt`.
    * **Format:** List all words in a **single line**, separated by commas (no spaces after the commas). **Do not use new lines or line breaks.**
    * **Case:** Please provide all words in lowercase unless the language specifically requires capitalization.
    * **Encoding:** Ensure the file is saved in **UTF-8** encoding to support special characters/scripts.
    * **Content:** Include only unique words; avoid duplicates within the same file.

4. Update the **Available Languages** table in the main [`README.md`](https://github.com/eymenefealtun/all-words-in-all-languages/blob/main/README.md):
	
	-   Increment the **Language Count** in the section heading 
    -   Add a new row to the table with the **English Name**, **Native Name**, **Word Count**, and a **link** to your new `.txt` file. (Insert your row into the table in **alphabetical order** based on the English Language Name. )   

5. If you collected the words from an external dataset, add the link to the **Sources** section at the bottom of the `README.md`.
 
 6. Submit a pull request.
